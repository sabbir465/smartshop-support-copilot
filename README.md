# SmartShop Support Copilot

An AI-powered customer support assistant that processes e-commerce refund requests using LLM tool calling and deterministic policy validation.

## Current Features

- FastAPI backend
- React + TypeScript frontend
- OpenAI function/tool calling
- Deterministic refund policy engine
- Customer chat interface
- Real-time reasoning timeline
- Refund decision dashboard

## Planned Improvements

- LangGraph orchestration
- SQLite CRM database
- SQLAlchemy ORM
- GitHub Actions CI/CD
- Docker
- Voice interaction
